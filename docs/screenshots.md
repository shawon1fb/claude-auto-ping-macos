# Screenshots

This page collects UI screenshots for the README. They are placeholders until
captured on a real machine with the Claude app installed.

## How to capture

1. Build and run the app (`./Scripts/build-release.sh`, then open it).
2. Open the menu bar popover and the Settings window.
3. Capture with `⌘⇧4` then Space (window capture), or `⌘⇧5`.
4. Save PNGs into `docs/images/` and reference them here and in the README.

## Needed shots

| File | What to show |
| --- | --- |
| `docs/images/menubar.png` | The popover: status, next run, interval, message, actions. |

Keep light and dark mode variants if possible, and avoid showing any private
content in the captured Claude window.
